Angumate

simple animations for angular.js

Why should I use it?

  • Animate ng-view now
  • Hardware accelerated
  • Cross-browser support
  • No animations as fallback
  • Small footprint

Quick Setup

  1. Download version 0.1 here
  2. include angumate in your html
  3. inject angumate in your routing module
  4. add angumate as promise to your routes
  5. add directives to your elements

How does it work?

  • Angumate listens for route changes
  • styles are applied to directives
  • if another route event is fired:
  • one deferred is created for all animations
  • which is resolved after animations have finished
